# Test repository

#### Initialize
```bash
git init
```

#### Add changed files
```bash
git add README.md
```

#### Make commit message
```bash
git commit -m "Initial Commit"
```

#### Set the remote repository
```bash
git remote add origin https://github.com/ygoreus/test1.git
```

#### Push changes to remote
```bash
git push -u origin master
```
